# SlackBot for PR's
This is a simple SlackBot that sends a notifcation when a pull request is made


## Requirements 

 - Create a Slack App 
 - Install WebHook 
 - Add a Secret to your Repo ( SLACK_WEBHOOK )
 - Have a Slack Account
